Уравнения

Решение уравнений первой степени с одним неизвестным

Для решения уравнений необходимо уметь:

  • а) освобождаться от знаменателя \(-\) записать уравнение в одну строчку;
  • б) из уравнения, записанного в строчку, определить нужный параметр.

1. Решение уравнения вида: \(x=\dfrac{a}{b}\):

  • \(x=\dfrac{a}{b} \Rightarrow bx=a\)
    Перенесем знаменатель в ту часть уравнения, где его нет

2. Решение уравнения вида: \(\dfrac{a}{b}=\dfrac{c}{x}\)

  • \(\dfrac{a}{b}=\dfrac{c}{x} \Rightarrow ax=bc\)
    Перемножим обе части уравнения крест на крест

3. Решение уравнения вида: \(ax=bc\)

  • \(ax=bc \Rightarrow \dfrac{a}{b}=\dfrac{c}{x}\)
    Поделим на то, что находится рядом с неизвестным

Квадратные уравнения

Уравнение вида \(ax^2 + bx + c = 0,\) где \(x -\)переменная, \(a, b, c -\) любые числа, причем \(a > 0\), называется квадратным.
Выражение \(D=b^2-4ac\) называется дискриминантом.

Корни квадратного уравнения находят по формуле: \(x_{1,2}=\dfrac{-b\pm \sqrt{D}}{2a}.\)

  • 1. \(D > 0;~ x_1=\dfrac{-b+\sqrt{D}}{2a};~ x_2=\dfrac{-b-\sqrt{D}}{2a},\) где \(х_1\) и \(х_2\) действительные различные числа;
  • 2. \(D = 0\), тогда \(x_1=x_2=-\dfrac{b}{2a};\)
  • 3. \(D < 0\) уравнение не имеет решений.

Неполные квадратные уравнения

Дано уравнение \(ax^2 + bx + c = 0\).

  • 1. Если при \(c = 0,~ ax^2 + bx = 0,\) тогда \(x(ax + b)= 0 \Rightarrow x_1 = 0;~ x_2 = \dfrac{b}{a}.\)
  • 2. Если при \(b = 0, ~ax^2 + c = 0 \Rightarrow ax^2 = -c \Rightarrow x^2 = -\dfrac{c}{a} \Rightarrow x_{1,2}=\pm\sqrt{-\dfrac{c}{a}},\) решение возможно, если \(\dfrac{c}{a}\le 0\).

Система уравнений

При решении задач по физике необходимо уметь решать системы двух или большего числа совместных уравнений.

При решении систем уравнений используются два основных способа:

  • а) способ алгебраического сложения;
  • б) способ подстановки.

В физике чаще всего используется способ подстановки, поэтому опишем его.

Дана система уравнений: \(\begin{equation*}\begin{cases}ax+by=c \\dx+ey=f \end{cases}\end{equation*}\)

Из первого уравнения выразим \(y\) через \(x\): \(y = \dfrac{c-ax}{b}\)

Подставим во второе уравнение: \(dx+e\dfrac{c-ax}{b}=f\)

Отсюда находим сначала \(x\), а затем \(y\):

\(x(d-\dfrac{ea}{b})=f-\dfrac{ec}{b} \Rightarrow x = \dfrac{fb-ec}{db-ea},~y=\dfrac{c-ax}{b}=\dfrac{c-a(\dfrac{fb-ec}{db-ea})}{b}=\dfrac{cd-af}{db-ea}.\)
